Given an array nums. We define a running sum of an array as runningSum[i] = sum(nums[0]â€¦nums[i])
Ans:
class Solution {
    public int[] runningSum(int[] nums) {
        int sum = 0;
        int [] runningSum = new int [nums.length];
        for ( int i =0; i<runningSum.length; i++)
        {
            sum+=nums[i];
            runningSum[i]=sum;
        }
        return runningSum;
    }
}
